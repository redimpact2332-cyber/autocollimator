# v8.6 印刷品質・座標整合修正版

- 印刷時の見出し背景を白、文字と枠を黒に固定
- ダークモード中でも帳票が黒く塗りつぶされないよう修正
- 左表の各行と右グラフの測定位置座標を完全に同じ比率で配置
- グラフの位置0から位置1を連続して描画
- 範囲線を「0～始点：破線／始点～終点：実線／終点～測定終端：破線」で描画
- 範囲線が途中で切れないよう印刷専用描画を修正
- 最大値が枠に接触しないよう左右に約25%の書込み余白を確保
- グラフ目盛を表ヘッダーと揃えた専用欄へ移動
- A4縦・1測定1ページ、4測定一括印刷、ブランク、任意2点赤線を維持
- v8.6 SAFE

注意：
iPhone/Braveの印刷プレビュー下部に出るURL・日時・ページ番号は
ブラウザ（AirPrint）が追加するヘッダー/フッターです。
WebアプリのCSSだけでは確実に消せません。
完全に消すには、今後「PDF帳票出力」機能を追加する方法が確実です。
