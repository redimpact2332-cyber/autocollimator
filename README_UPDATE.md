# autocollimator v7.3 SAFE

このZIPは v7.2 SAFE を土台にした最新版です。

## v7.3 変更点
- 表の各測定セルをタップした瞬間に、選択中の測定1〜4へ計算結果とグラフを即時切替
- 測定1/2/3/4の切替ボタンでも計算結果とグラフを即時更新
- service-worker cache を `autocollimator-v7-3-select-update` に更新

## 使い方
GitHub Desktopで既存リポジトリのフォルダへ中身を上書きし、Commit → Pushしてください。

## 注意
`.git` フォルダは消さないでください。
